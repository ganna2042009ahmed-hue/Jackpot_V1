# Jackpot_V1

## Web description

Navigation Bar:

Includes links to: Home, About, Menu, Gallery, and Contact
Displays the restaurant logo for branding

Home Section:

Welcoming header with the restaurant name
Short introduction paragraph
“Book a Table” button for reservations

About Section:

Information about the restaurant’s history
Highlights special features and unique aspects
Includes images related to the restaurant
“Learn More” button for additional details

Menu Section:

Displays available dishes with:
Description
Price
Includes a Special Offers page:
Shows discounted items
Displays price before and after discount
Includes images of dishes

Gallery Section:

Collection of images related to the restaurant
Includes timing information (lunch and dinner hours)

Contact Section:

Restaurant address
Working hours
Contact form for sending messages

Technologies Used:
HTML
CSS

Usage:

Browse the website using the navigation bar
View menu items and special offers
Explore images in the gallery
Contact the restaurant using the form

License:

This project is created to help restaurants improve their online presence.
